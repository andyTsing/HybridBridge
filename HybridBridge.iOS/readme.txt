      Soroush Falahati's HybridBridge Library
------------------------------------------------------
This package is the iOS platform package of the HybridBridge library, expanding its  functionalities
to the iOS environment.
Using this library you can simply connect the UIWebView control with your controller class:

new UIWebViewHybridBridge(webView).Add(new ClassBridge<MethodSamplesController>());

For more information about this project and usage samples, please visit the project's GitHub page at
https://github.com/falahati/HybridBridge
