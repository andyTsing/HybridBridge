      Soroush Falahati's HybridBridge Library
------------------------------------------------------
This is an expanding package to the HybridBridge project letting platform implementations to
create a local HTTP server and handle the request. For this library to be usable you need to
add one of the following packages next to it:

* HybridBridge.UWP
* HybridBridge.Win
* HybridBridge.Win81

Or you can expand the `HybridBridge.WebServer.LocalBridgeServer` class for your environment.

For more information about this project and usage samples, please visit the project's GitHub
page at https://github.com/falahati/HybridBridge
